# nuxt-layer-basic

A Nuxt layer that bundles shared client/server utilities and UI-agnostic components for use across Nuxt projects.

## Installation

```bash
pnpm add @bjj/nuxt-layer-basic
```

Peer dependencies (install in the host project):

```
nuxt vue
```

## Usage

Extend the layer from the host project's `nuxt.config.ts`:

```ts
export default defineNuxtConfig({
  extends: ['@bjj/nuxt-layer-basic'],
})
```

The layer auto-registers components, app utilities, and Nitro server utilities so the symbols become globally available in the consumer.

> Note: Tailwind utility classes you see in story wrappers are loaded only by `.playground` for demo purposes — components themselves ship without Tailwind.
